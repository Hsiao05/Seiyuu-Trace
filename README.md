# 声迹（Seiyuu-Trace）

声迹是一个基于 Bangumi 数据的动画声优统计网站。用户可以通过导入自己的 Bangumi 动画收藏，快速整理已经看过的作品，并统计自己听过哪些声优、看过哪些角色、最熟悉哪些声音。

这个项目目前是一个纯前端静态网站，页面文件位于 `outputs/` 目录，可以直接部署到 GitHub Pages 等静态托管平台。

## 功能

- 根据 Bangumi UID 导入已完成收藏的动画片单
- 按作品 ID 批量导入动画
- 通过 Bangumi API 搜索作品
- 导入时显示读取进度，并支持取消
- 统计看过的动画数量、听过的声优数量和角色记录数量
- 将角色区分为主角、配角、客串、闲角
- 提供加权排行和无权排行两种统计方式
- 支持查看每位声优对应的作品和角色明细
- 支持跳转到 Bangumi 的声优、作品和角色页面
- 提供“声迹颁奖礼”总结页，展示总览数据和排名回顾
- 默认数据为空，不会预置演示片单或演示声优
- 兼容桌面端和移动端浏览
